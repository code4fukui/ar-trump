# AR trump

AR trumpは、52枚のトランプカードをシャッフルして3Dで表示する拡張現実（AR）アプリケーションです。

## デモ
[GitHub Pages](https://code4fukui.github.io/ar-trump/)でライブデモをご覧いただけます。

## 機能
- 52枚のトランプカードをシャッフルして3Dで表示
- 表と裏で異なるテクスチャを持つ両面カードをサポート
- 回転や移動のエフェクトでカードをアニメーション化

## 必要条件
このプロジェクトは`egxr.js`ライブラリを使用しており、依存関係として含まれています。

## 使い方
アプリケーションを実行するには、ウェブブラウザで`index.html`ファイルを開いてください。
