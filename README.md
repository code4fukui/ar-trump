# AR trump

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

AR trump is an augmented reality application that displays a shuffled deck of 52 playing cards in 3D.

## Demo
You can view the live demo at [GitHub Pages](https://code4fukui.github.io/ar-trump/).

## Features
- Displays a shuffled deck of 52 playing cards in 3D
- Supports double-sided cards with different front and back textures
- Animates the cards with rotating and shifting effects

## Requirements
This project uses the `egxr.js` library, which is included as a dependency.

## Usage
To run the application, open the `index.html` file in a web browser.
